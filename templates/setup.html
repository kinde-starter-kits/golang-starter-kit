Configuration Required

Let's get your Kinde credentials set up!

‚ö†Ô∏è Important: The application requires valid Kinde credentials to run. This should only take 2-3 minutes to set up!

Step 1: Get Your Kinde Credentials

  1. Go to app.kinde.com and sign in
  2. Navigate to Settings ‚Üí Applications
  3. Select or create an application
  4. Copy these values:
    • Domain (e.g., https://yourbusiness.kinde.com)
    • Client ID
    • Client Secret (click "Show" to reveal)

Step 2: Configure Callback URLs in Kinde

In your Kinde application settings, add these URLs:

  • Allowed callback URLs: http://localhost:3000/callback
  • Allowed logout redirect URLs: http://localhost:3000

Click Save

Step 3: Edit Your .env File

Open this file in your text editor:

.env

Update it with your Kinde credentials:

KINDE_DOMAIN=https://yourbusiness.kinde.com KINDE_CLIENT_ID=your_client_id_here KINDE_CLIENT_SECRET=your_client_secret_here KINDE_REDIRECT_URI=http://localhost:3000/callback KINDE_LOGOUT_REDIRECT_URI=http://localhost:3000 PORT=3000 SESSION_SECRET=any-random-long-string-here
Important: Replace your_client_id_here, your_client_secret_here, and yourbusiness with your actual values!

Step 4: Restart the Server

After saving your .env file:

  1. Stop the server (press Ctrl+C in the terminal)
  2. Start it again:
go run main.go
